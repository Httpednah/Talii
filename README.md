# Talii – Local Experience Booking Platform


**Talii** is a web application for discovering, browsing, and booking local experiences. It helps users explore unique experiences in their city, from cultural activities to adventurous outings, with an easy-to-use interface and smooth navigation.

---

## Features

- Browse experiences by category (Food, Adventure, Culture, etc.)  
- View detailed information for each experience  
- Book experiences (MVP: simulated booking functionality)  
- Responsive design for desktop and mobile  
- Subtle, pleasant theme with soft color accents  

---


## Technologies Used

- **React** – Frontend library  
- **Vite** – Development server and build tool  
- **React Router** – Client-side routing  
- **CSS** – Styling with subtle, theme-consistent colors  
- **Node.js & npm** – Package management
